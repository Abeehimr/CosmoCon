Web Hackathon

Build. Innovate. Transform.

About the Hackathon

The Web Hackathon at CosmoCon'25 is a 24-hour marathon of creativity, innovation, and coding excellence. Teams will conceptualize, design, and develop fully functional web applications that solve real-world problems or push the boundaries of web technology.

Whether you're passionate about social impact, cutting-edge tech, or creative storytelling through web experiences, this hackathon provides the perfect platform to showcase your skills. Mentors will be available throughout to guide and support your journey from idea to deployment.

Event Information

Registration Fee

$25 per team

Team Size

2-4 members

Duration

24 hours

Format

Hybrid

Hackathon Rules

  • Teams must consist of 2 to 4 members
  • All team members must register before the event begins
  • Projects must be started from scratch during the hackathon period
  • Use of existing libraries, frameworks, and APIs is allowed
  • Projects must be web-based and accessible via a URL
  • All code must be pushed to a public GitHub repository
  • Teams must present their project in a 5-minute pitch to judges
  • Judging criteria: Innovation (30%), Technical Implementation (30%), Design (20%), Presentation (20%)
  • Plagiarism or use of pre-existing projects will result in disqualification
  • Teams must use ethical coding practices and respect API usage limits
